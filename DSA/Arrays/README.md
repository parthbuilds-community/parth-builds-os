## Array Basics

Arrays are linear data structures that store elements in contiguous memory locations.

### Common Operations
- Traversal
- Insertion
- Deletion
- Searching
- Sorting

### Time Complexity

| Operation | Complexity |
|-----------|------------|
| Access | O(1) |
| Search | O(n) |
| Insertion | O(n) |
| Deletion | O(n) |
