# 3 Projects That Teach More Than Tutorials

## 1. Personal Portfolio Website
**What you'll learn:**
- HTML structure and semantic tags
- CSS flexbox/grid for layouts
- JavaScript for interactivity
- Deployment process

**How to start:**
- Use a template or start from scratch
- Include: About, Projects, Contact sections
- Make it responsive for mobile
- Deploy on GitHub Pages (free)

## 2. To-Do App with Database
**What you'll learn:**
- CRUD operations (the foundation of most apps)
- Frontend-backend communication
- Database basics (SQL or MongoDB)
- User authentication

**Feature ideas:**
- Add, edit, delete tasks
- Mark tasks complete
- Filter by status
- User accounts (advanced)

## 3. Automation Script
**What you'll learn:**
- Reading/writing files
- Using libraries and APIs
- Solving real problems with code
- Task automation

**Script ideas:**
- Bulk rename files based on pattern
- Download images from a webpage
- Organize files by extension
- Send automated emails/messages

## Tips for All Projects
- Start minimal, add features one at a time
- Google every error you hit
- Push to GitHub as you build
- Write a README explaining what you learned
